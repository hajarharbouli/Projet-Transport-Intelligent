# Projet : Système de Transport Intelligent

Projet Java utilisant plusieurs **patrons de conception** :
- Factory
- Builder
- Adapter
- Decorator
- Composite
- Mediator
- Observer
- Facade
- Chain of Responsibility

## Auteurs
Hajar Harbouli – Étudiante en Génie Informatique  GI21
Encadré par : [madame hallou amal]

## Description
Cette application simule un système de transport intelligent intégrant différents types de véhicules (Bus, Taxi, Tramway, Vélo) avec des capteurs, un centre de supervision et un service de paiement.

## Exécution
Exécuter la classe `Main.java` dans le package `main`.
